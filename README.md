<h1 class="code-line" data-line-start=0 data-line-end=1 ><a id="2019_0"></a>2019年的新维加斯入门指南</h1>
<p class="has-line-data" data-line-start="1" data-line-end="3"><img src="https://imgsa.baidu.com/forum/w%3D580/sign=d7c34610f936afc30e0c3f6d8318eb85/fa23ecdeb48f8c544062ceff34292df5e1fe7f1b.jpg" alt="intro" title="intro"><br>
<strong>2019年了，新维加斯不应该是这个游戏体验</strong></p>
<p class="has-line-data" data-line-start="4" data-line-end="7">本指南旨在建立一种一站式的入门指导，帮助你了解游玩新维加斯需要的准备工作。并整合最新的优化配置选项，使你以最快的速度以及最丝滑的方式体验废土的旅程。<br>
本指南不仅适合刚接触新维加斯的新手，也适合远离沙场已久，准备回归的老兵。因为一些Mod和优化方式的发展已与之前的方式不相兼容，因此本指南将会是你最好的选择。<br>
本指南内容主要基于外网最近比较全面的一个MOD指南Viva New Vegas，但并非单纯将其汉化。本指南会根据中文补丁的安装修改一些流程。因此如果你想游玩英文版，建议你直接阅读原指南，原址<em><a href="http://vivanewvegas.github.io">vivanewvegas.github.io</a></em></p>
<p class="has-line-data" data-line-start="8" data-line-end="9">首先说一下结论，我使用一台笔记本电脑，配置为</p>
<ul>
<li class="has-line-data" data-line-start="9" data-line-end="10">CPU：I7-5500u</li>
<li class="has-line-data" data-line-start="10" data-line-end="11">内存：8GB 1666Mhz DDR3</li>
<li class="has-line-data" data-line-start="11" data-line-end="12">显卡：GTX950m 2G DDR3</li>
<li class="has-line-data" data-line-start="12" data-line-end="13">硬盘：128GB SATA3.0 SSD</li>
<li class="has-line-data" data-line-start="13" data-line-end="15">系统：Win10 LTSC 2019</li>
</ul>
<p class="has-line-data" data-line-start="15" data-line-end="17">目前的配置及优化可以使原版游戏在1080P，最高画质下以60帧稳定运行。即使加了大量画质MOD后依旧可以在高画质，900P的条件下运行5个小时不跳出。我觉得这样的游戏体验才是2019年的新维加斯应该有的。<br>
并且必须一提的是，网上流传了很久新维加斯不适合在Win10上运行。以目前的情况看已经没有那么不堪，但前提是你的Win10版本不能太旧，也不能太新。我的LTSC是基于1809的，可以完美运行。</p>
<p class="has-line-data" data-line-start="18" data-line-end="22">一、一般信息<br>
在下载前，请你先确保自己的配置足够运行新维加斯。并且与一般游戏不同的是，这里建议你不仅要满足官方给出的最低要求，最好是能接近推荐的配置要求。一方面作为一款9年前的游戏，如果只以较低画质运行，其画面表现力将十分不堪。更重要的是新维加斯在低画质选项下曾产生了很多BUG，一些MOD也会需要你将贴图质量调到高以上。因此建议你在购买下载前关注一下配置要求。<br>
当然通过本指南，你可以最大化发挥硬件的资源，改善原版游戏优化糟糕的现象，提升运行的稳定性与流畅度，所以只要你的配置不是太差，莫哈维废土都可以任你驰骋。<br>
要声明的是，本指南初期只针对原版游戏，不会添加或修改游戏内容。如果你是你想通过MOD改变游戏玩法或内容，抑或是需求画质的提升，请等待后期我的更新，或直接阅读原MOD指南。</p>
<p class="has-line-data" data-line-start="23" data-line-end="28">二、准备工作<br>
在运行游戏之前，你需要什么？当然是游戏本身。不论网上流传着什么版本，我在这里只推荐Steam或GOG的终极版，也就是全DLC的版本。纯净的正版是本指南适用的唯一版本，如果是保证完全纯净的学习版应该也可以，但我不能保证其他版本也有效果。<br>
除此之外你还需要一个MOD管理器，也许对于本入门指南来说，MOD管理器并没有发挥太大左右，但熟悉MOD管理器将为你未来的游戏旅程很有帮助。本指南使用Mod Organizer 2，如果你之前没有使用过或使用较老的MOD管理器，建议你更新到Mod Organizer 2，这是目前最先进的MOD管理器，拥有很多优良的特性。<br>
因为本指南专门针对想玩中文的玩家，你肯定还需要一个中文补丁。目前只推荐民间汉化第5版，这是在天邈汉化出来前(虽然都不知道还到底会不会出来)最完善的，也是安装最简单的一个版本。注意它只支持终极版或者升级了UP7并且安装了所有dlc的版本。<br>
其他需要的准备工作还有：最新的显卡驱动，足够的硬盘空间以及了解INI文件(实际上就是文本文档)的修改。</p>
<p class="has-line-data" data-line-start="29" data-line-end="31">三、系统设置<br>
这一部分主要针对Win10系统，Win7可以跳过。</p>
<ul>
<li class="has-line-data" data-line-start="31" data-line-end="32">首先关闭系统自带的游戏模式，对于新维加斯来说该模式会造成不稳定</li>
</ul>
<ul>
<li class="has-line-data" data-line-start="34" data-line-end="35">其次调整你的输入法，Win10自带的输入法影响了很多游戏，如果你不能保证游戏过程中不会切换到中文，最好的方法是单独添加英语作为第二键盘，并按alt+shift切换。</li>
</ul>
<ul>
<li class="has-line-data" data-line-start="37" data-line-end="39">最后是关掉通知，以及其他可能在后台弹出对话框/提示的软件。新维加斯在切换窗口时卡死可能会让你疯掉，如果你必须要频繁切回桌面，后面针对此会有单独的步骤，但总的来说建议在游玩新维加斯的过程中尽量不要切出去。</li>
</ul>
<p class="has-line-data" data-line-start="39" data-line-end="40">如果你使用笔记本电脑，请确保电源选项选择正确，如<strong>高性能</strong>或<strong>卓越性能</strong></p>
<p class="has-line-data" data-line-start="41" data-line-end="44">四、游戏安装<br>
1.安装参数：<br>
所有modding工具必须安装在所有默认Windows文件夹之外（例如Program Files，Program Files（x86）和Documents）。Fallout New Vegas和Mod Organizer 2必须安装在同一个盘上，但不要将Mod Organizer 2和mod文件夹安装在Fallout New Vegas根目录下。</p>
<p class="has-line-data" data-line-start="45" data-line-end="49">2.常用术语：<br>
根文件夹：<em>Steam / steamapps / common / Fallout New Vegas</em> 或 <em>GOG Galaxy / Games / Fallout New Vegas</em><br>
Data文件夹：<em>Steam / steamapps / common / Fallout New Vegas / Data</em> 或 <em>GOG Galaxy / Games / Fallout New Vegas / Data</em><br>
MO2：Mod Organizer 2（将很快安装）</p>
<p class="has-line-data" data-line-start="50" data-line-end="53">3.<strong>禁用叠加层</strong>：<br>
需要为Fallout New Vegas禁用任何叠加以防止内存泄漏。一些具有覆盖的常见应用程序是Steam，GOG，NVIDIA GeForce Experience，Discord和MSI Afterburner。<br>
如Steam的禁用方法为：在库中右键点击Fallout：New Vegas，选择 属性，在 常规 选项卡下找到 在游戏中启动Steam界面 并反勾选它。这一步会使你无法使用Steam自带的截图功能，完成成就时也不会有右下角弹窗，但是还是非常建议你不要启动这一项，新维加斯游戏内有成就提示，后面也会介绍其他截图方式。</p>
<p class="has-line-data" data-line-start="54" data-line-end="58">4.设置Fallout New Vegas Modding文件夹：<br>
您的modding文件夹将充当您安装的所有工具的主目录，以及任何备份/屏幕截图等。从现在开始，此文件夹将被称为Modding文件夹。<br>
在与Fallout New Vegas安装相同的盘上的默认Windows文件夹之外的所选位置创建一个新文件夹，注意在根文件夹夹之外。将它命名为任何你喜欢的，如 <em>Fallout NV Modding</em> ，并创建两个子文件夹名为 <em>Backups</em> 和 <em>Tools</em><br>
备份整个游戏，将Fallout New Vegas文件夹复制到刚才创建的Backups中</p>
<p class="has-line-data" data-line-start="59" data-line-end="62">5.生成新鲜的.INI文件：<br>
从根文件夹运行FalloutNVLauncher.exe<br>
弹出 Detecting Video Hardware 的窗口，单击确定。之后弹出窗口提示程序根据你的硬件已设置画质，单击确定。</p>
<ul>
<li class="has-line-data" data-line-start="62" data-line-end="64">如果没有任何弹出窗口，请导航到 <em>文档/My Games/FalloutNV</em> 并删除以.INI结尾的所有文件，然后重新运行FalloutNVLauncher.exe</li>
</ul>
<p class="has-line-data" data-line-start="64" data-line-end="69">选择 Options<br>
确认游戏检测到并正确使用你的GPU，并选择了合适画质。如果没有正确识别GPU，请手动选择并调整画质选项，最好以Medium或High为初始选项。<br>
根据你的偏好设置分辨率（启动器通常将其设置为低于最大分辨率的一个）<br>
注意 Antiailiasing 项可以设置为8 Samples（启动器最多将其设置为4 Samples）<br>
单击 OK，然后单击 EXIT 退出</p>

<p class="has-line-data" data-line-start="219" data-line-end="222">八、使用BethINI优化管理INI文件<br>
BethINI试图统一由Bethesda创建的游戏的INI配置文件，同时优化您的游戏设置以获得最佳图形保真度和性能。它目前支持Oblivion，Skyrim，Skyrim特别版，Fallout 3，Fallout New Vegas和Fallout 4.<br>
使用BethINI有许多优点：</p>
<ul>
<li class="has-line-data" data-line-start="222" data-line-end="223">它会自动将您的INI文件重新排序为合理的顺序，使其更容易导航以进行手动调整。</li>
<li class="has-line-data" data-line-start="223" data-line-end="224">它自动修复常见错误，包括一些可能导致CTD的错误。</li>
<li class="has-line-data" data-line-start="224" data-line-end="225">它会自动检测您已安装的mod并在适当的情况下进行mod作者推荐的更改。</li>
<li class="has-line-data" data-line-start="225" data-line-end="226">它的预设在图形保真度和性能方面（95％的时间）优于通过游戏启动器创建的官方预设。</li>
<li class="has-line-data" data-line-start="226" data-line-end="227">它允许修改比通过本机游戏设置可访问的更多设置，并提供每个设置及其功能的说明。</li>
<li class="has-line-data" data-line-start="227" data-line-end="229">它会自动备份您的INI文件。</li>
</ul>
<p class="has-line-data" data-line-start="229" data-line-end="232">注意事项：<br>
BethINI standalone是一个便携式应用程序。你可以在任何地方运行它，简单的解压并运行。不幸的是，独立版本很容易被防病毒软件标记为误报，所以最好请其添加到将杀毒软件的白名单中。<br>
请注意，不要通过Mod Organizer 2运行，也不要在Mod Organizer 2打开时运行BethINI。</p>
<p class="has-line-data" data-line-start="233" data-line-end="243">安装使用说明：<br>
1.运行前请确保已运行过FalloutNVLauncher.exe并生成了相应的INI文件<br>
2.运行BethINI.exe<br>
3.在弹出的窗口中下拉选择 Fallout New Vegas<br>
4.进入程序后导航到 Setup 选项卡，<strong>反选</strong>全部5个选项方框。之后在 Mod Organizer 一栏下拉菜单中选择 Browse ,在弹出的框中导航到 Mod Organizer 2 文件夹并双击选择 ModOrganizer.exe<br>
5.在 INI Path 一栏下拉菜单中选择您现在使用的Mod Organizer 2配置文件，如 ModOrganizer&gt;Defaul 。在弹出的对话框中选择 NO ，此时BethINI会重新启动，等待其重启即可。<br>
6.重新启动BethINI后导航到 Basic 选项卡，在 Presets 栏中选择 BethINI Presets ，再选择 Recommended Tweaks ，程序会自动优化相应的INI文件，之后请根据配置在 Poor 到 Ultra 中选择合适的画质选项。<br>
7.选择完画质后在 Display 一栏中选择合适的分辨率，如果没有需要的分辨率则可以手动输入。可以自行选择 Antialiasing 和 Anisotropic Filtering 倍数(一般使用预设即可)。在右边选择 Allow 3.0 Shaders ，其他保持不变。<br>
8.最后选择下方的 Enable File Selection ，确定一切选择正确后点击 Save and Exit 退出。等待程序关闭后导航到 <em>文档/My Games/FalloutNV</em> 打开 RendererInfo.txt 。查看下方 Shader Package 后的数字，如果是19，则不需要进一步修改。如果不是19，请导航到 <em>data/Shaders</em> 文件夹找到序号对应Shader Package后数字的文件，如 shaderpackage013.sdp ，将其命名为 shaderpackage013.sdp.backup 。<br>
之后把 shaderpackage019.sdp 复制一份到其他位置，将复制的shaderpackage019.sdp重命名为shaderpackage013.sdp，再剪切回原文件夹。整个过程即是备份正在使用的shaderpackage，并将shaderpackage019.sdp重命名以强制启用。</p>
<blockquote>
<p class="has-line-data" data-line-start="243" data-line-end="244">强制启用shaderpackage019以打开Shaders 3.0，这应该会增加游戏帧数及稳定性。</p>
</blockquote>
